SimpleDataCollection Extension

Description:

This sample demonstrates how to customize the data collection workflow.
Products: ArcGIS Mobile

Platforms: Windows

Requires: Visual Studio 2008 (SP1), .NET 3.5 (sp1)

How to use:
  1. Open the project within Visual Studio 2008. Open SimpleDataCollectionExtension.cs file, replace the layer name there with a layer name from your mobile cache.
    To learn how to setup your development machine, click here.
  2. Compile the project and deploy assembly to the ArcGIS mobile Windows application bin folder of the work machine. Click here for more information for deployment.
  3. Build and deploy Mobile Project assembly file (comes with the sample project) to <ArcGIS Mobile Install Path>\bin\Extensions folder on your desktop where you will author your mobile project with Mobile Project Center application. In Mobile Project Center application, add "SimpleDataCollectionExtension" to your project in Capabilities tab. Deploy your mobile project. For more details on deployment, click here.
  4. Start the Windows Application.
  5. open the mobile project.
  6. Notice that the data collection workflow is simplied.

 

Where to get the sample:

The sample is available from the following folder:

<ArcGIS_Mobile_Install_Path>\DeveloperKit\Samples\ApplicationFramework\Windows
